# Application up and running for use this repo
### docker build 
```
 docker build .
 docker images
 docker build . -t cal-python
 ```

 ### docker images check
 ```
 docker images
 ```

 ### Docker application check & running 
 ```
 docker ps
 ```

### docker run the application

```
docker run -p 5000:5000 -d cal-python
```
